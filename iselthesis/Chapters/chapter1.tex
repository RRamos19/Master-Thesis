\chapter{Introdução}
\label{capitulo:introdução}

A elaboração de horários de cursos universitários é tipicamente feita de forma manual e todos os anos. Devido à existência de múltiplos horários possíveis mas não ótimos torna esta atividade muito demorada e complexa. Em certos casos não é possível criar um horário possível devido ao número de restrições presentes.

No momento de concretização de um horário são apresentadas múltiplas restrições. As restrições podem ser \textit{hard} ou \textit{soft}. Restrições \textit{hard} são de cumprimento obrigatório não sendo possível criar um horário caso a restrição seja violada. Restrições \textit{soft} podem não ser cumpridas apesar de haver um incentivo para o cumprimentos das mesmas, caso não sejam cumpridas o horário não deixa de ser válido. A qualidade de um horário é quantificado relativamente à quantidade de restrições \textit{soft} violadas.

O projeto visa o desenvolvimento de uma aplicação gráfica que permita a criação de horários apenas necessitando uma configuração inicial acerca do ciclo (licenciatura, mestrado) do horário e os dados relativos às salas, disciplinas e professores. Para a criação dos horários será utilizada a meta heurística \acrfull{sa} (e será seguido o padrão apresentado no concurso \acrfull{itc} 2019 CONFIRMAR).
