\section{Casos de utilização}

Nesta secção serão apresentados os casos de utilização do projeto. Todos os casos de utilização têm um ator em comum sendo este o utilizador da aplicação.

\subsection{Inserção de dados através de um ficheiro}
Resumo: Leitura de um ficheiro, que siga o padrão do concurso \glsxtrshort{itc} 2019, e armazenamento dos dados numa base de dados.

Cenário principal:

\begin{enumerate}
    \item O caso de utilização inicia-se quando o ator pressiona o botão de inserção de dados através de ficheiro
    \item O ator escolhe o ficheiro que deve ser lido por parte da aplicação
    \item A aplicação processa os dados e guarda numa base de dados a informação \label{casos_util:caso_1_process_dados}
    \item O caso de utilização termina
\end{enumerate}

Cenário alternativo 1:

\begin{enumerate}
    \item No passo \ref{casos_util:caso_1_process_dados} o ficheiro escolhido não segue o padrão do concurso \glsxtrshort{itc} 2019.
    \item Apresentação de um erro ao utilizador
    \item O caso de utilização termina
\end{enumerate}
